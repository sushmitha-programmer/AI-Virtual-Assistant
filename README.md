# 🤖 AI Virtual Assistant

An intelligent AI Virtual Assistant built with Python that performs voice-controlled tasks such as searching the web, opening applications, telling the time, and responding to user commands.

## 📌 Features

- 🎤 Voice Recognition
- 🔊 Text-to-Speech
- 🌐 Google Search
- ▶️ YouTube Search
- 📖 Wikipedia Search
- 🕒 Current Date & Time
- 😂 Tell Jokes
- 🌦️ Weather Information
- 📂 Open Desktop Applications
- 🧮 Basic Calculations
- ❌ Exit Using Voice Command

## 🛠️ Technologies Used

- Python 3
- SpeechRecognition
- pyttsx3
- pywhatkit
- Wikipedia
- PyAudio
- OpenCV *(optional for future upgrades)*

## 🚀 Installation

1. Clone the repository:

```bash
git clone https://github.com/sushmitha-programmer/AI-Virtual-Assistant.git
```

2. Go to the project folder:

```bash
cd AI-Virtual-Assistant
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the assistant:

```bash
python main.py
```
